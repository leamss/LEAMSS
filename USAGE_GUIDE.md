# LEAMSS Immigration Portal - Usage Guide

## Getting Started

### Accessing the Portal
1. Open your browser and navigate to the application URL
2. You'll see the login page with demo credentials displayed
3. Login with appropriate role credentials

---

## Admin Portal Guide

### Dashboard
- View pending sales awaiting approval
- Monitor active cases count
- Track total revenue

### Managing Sales
1. Click **"Pending Sales"** in sidebar
2. Review each sale submission:
   - Client details
   - Product selected
   - Payment information
   - Partner name
3. To approve:
   - Select a Case Manager from dropdown
   - Click **"Assign & Approve"**
   - System automatically creates client account and case
4. To reject:
   - Click **"Reject"** button

### Product Management
1. Click **"Products"** in sidebar
2. **Create New Product**:
   - Enter product name (e.g., "Australia PR - Skilled Route")
   - Set fee amount
   - Set commission rate (%)
   - Add description
   - Click **"Create Product"**

3. **Add Workflow Steps**:
   - Select product from dropdown
   - Enter step name (e.g., "Document Verification")
   - Set step order (1, 2, 3...)
   - Click **"Add Workflow Step"**
   - Repeat for all workflow stages

### User Management
1. Click **"Users"** in sidebar
2. **Create New User**:
   - Enter name and email
   - Set password
   - Enter mobile number
   - Select role (Case Manager, Partner, Admin)
   - Click **"Create User"**

### Viewing All Cases
1. Click **"All Cases"** in sidebar
2. See complete list of cases with:
   - Case ID
   - Client name
   - Product
   - Assigned case manager
   - Current workflow step

## Partner Portal Guide

### Dashboard
- View total sales count
- Monitor approved sales
- Track total commission earned
- See recent sales list

### Creating a New Sale
1. Click **"New Sale"** button
2. Fill in client information:
   - Client name
   - Client email
   - Client mobile number
3. Select product from dropdown
4. Enter payment details:
   - Fee amount (auto-filled from product)
   - Amount received
   - Payment method
   - Payment reference number
5. **Upload Required Documents**:
   - Payment Receipt (PDF/JPG/PNG)
   - Signed Agreement
   - Client Passport
6. Click **"Create Sale"**
7. Wait for admin approval

### Viewing Sales
1. Click **"My Sales"** in sidebar
2. See all your submitted sales with:
   - Status (pending, approved, rejected)
   - Client details
   - Product information
   - Commission amount (for approved sales)

### Tracking Commission
1. Click **"Commission"** in sidebar
2. View total commission earned
3. See breakdown by sale:
   - Client name
   - Product
   - Commission rate and amount
   - Sale date

## Case Manager Portal Guide

### Dashboard
- View total assigned cases count
- Monitor pending documents for review
- See recent cases list

### Managing Cases
1. Click **"My Cases"** in sidebar
2. Click on any case card to view details
3. **Case Information Section**:
   - Client name and email
   - Product type
   - Partner name

### Workflow Management
1. In case details, view **"Workflow Steps"** section
2. Each step shows:
   - Step name
   - Current status
   - Notes
3. **Update Step Status**:
   - Click status dropdown for any step
   - Select: Pending, In Progress, or Completed
   - Status updates automatically

### Document Review
1. Scroll to **"Documents"** section in case details
2. View all uploaded documents:
   - Filename
   - Step name
   - Upload date
   - Current status
3. **Review a Document**:
   - Click **"Review"** button on pending documents
   - Select status:
     - **Approve**: Document accepted
     - **Reject**: Document not acceptable
     - **Revision Required**: Client needs to resubmit
   - Add review comment
   - Click **"Submit Review"**

## Client Portal Guide

### Case Overview
- View your case ID and product type
- See progress bar showing completion percentage
- View current workflow step
- See completed vs pending steps count
- View assigned case manager information

### Workflow Checklist
1. Review the step-wise checklist
2. Each step shows:
   - ✅ Green for completed
   - 🔵 Blue for in progress
   - ⚠️ Amber for pending
3. Read notes from case manager for guidance

### Uploading Documents
1. Go to **"Upload Documents"** section
2. **Select Step**: Choose workflow step from dropdown
3. **Choose File**: Click to select file (PDF, JPG, or PNG)
4. Click **"Upload Document"**
5. Document status will show as "pending_review"

### Viewing Document Status
1. Scroll to **"My Documents"** section
2. Each document shows:
   - Filename and step
   - Upload date
   - Review status:
     - **Pending Review**: Awaiting case manager review
     - **Approved**: Document accepted
     - **Rejected**: Document not accepted
     - **Revision Required**: Need to upload again
   - Case manager comments (if any)

## Tips & Best Practices

### For Partners
- Always upload all 3 mandatory documents with each sale
- Use clear payment reference numbers for easy tracking
- Double-check client email addresses for portal access
- Monitor commission dashboard regularly

### For Case Managers
- Update workflow steps promptly to keep clients informed
- Provide clear, helpful comments when reviewing documents
- Check for pending documents regularly
- Add detailed notes to workflow steps for client guidance

### For Clients
- Upload high-quality, clear document scans
- Follow case manager instructions carefully
- Check document status regularly for feedback
- Keep documents organized by workflow step

### For Admins
- Create detailed workflow steps for each product
- Assign cases to case managers based on workload
- Monitor pending sales daily
- Create comprehensive product descriptions
- Set realistic commission rates

## Troubleshooting

### Cannot Login
- Verify email and password are correct
- Check caps lock is off
- Use demo credentials if testing

### Sale Not Appearing
- Ensure all required fields are filled
- Check all 3 documents are uploaded
- Refresh the page

### Document Upload Failed
- Check file size (max 10MB)
- Ensure file type is PDF, JPG, or PNG
- Try a different file

### Case Not Visible
- Client: Wait for admin to approve sale and create case
- Case Manager: Ensure case is assigned to you
- Check you're logged in with correct role

### Step Status Not Updating
- Case Managers: Ensure you selected status from dropdown
- Refresh the page to see latest changes
